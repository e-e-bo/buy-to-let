# buy-to-let
Code for the paper Buy to let
* Work in progress *

Prep:
Code for data preparation (Stata).
- The .sas files are only of use for preparing micro data from Statistics Norway.

Data:
Aggreagate statistics from the Statistics Norway statbank (www.ssb.no/statbank; in Excel format).
- Antbolig_full_ex.xlsx has data from Table 06265 (downloaded 13.03.2024).
- Boligbygg_ex_xlsx has data from Table 05889 (downloaded 13.03.2024).
- Cpimall.xlsx has data from Table 03013 (downloaded 26.08.2016).
- Mortgage rates_ex.xlsx has data from Table 07045 and Table 07200 (downloaded 24.01.2024).

The micro data used in this paper are confidential tax and other administrative records from Statistics Norway. I am not authorized to share the original data sets for confidentiality reasons. For researchers interested in replicating the results, information on how to obtain the data from Statistics Norway is available at https://www.ssb.no/en/data-til-forskning/utlan-av-data-til-forskere. The author can be contacted for assistance with this process, if required.
